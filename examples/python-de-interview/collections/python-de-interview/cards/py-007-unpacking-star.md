> Explain unpacking with `*` in assignment.

---

```python
first, *rest = [1, 2, 3, 4]
# first=1, rest=[2, 3, 4]

a, *mid, b = [1, 2, 3, 4]
# a=1, mid=[2, 3], b=4
```
