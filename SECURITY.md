# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, use one of the following:

1. [GitHub private vulnerability reporting](https://github.com/OmarFarooq908/deckflow/security/advisories/new) (preferred)
2. Email the maintainer via the contact listed on the GitHub profile

Include:

- Description of the issue and potential impact
- Steps to reproduce
- Affected version(s)
- Suggested fix (if any)

You should receive an acknowledgment within 72 hours. We will work with you on a fix and coordinated disclosure timeline.
